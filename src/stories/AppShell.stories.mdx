# AppShell

Root component of the application. Thanks to it you can control the theme of the app.

<Props of={AppShell} />

## Usage

```jsx
import { AppShell, Button } from '@zsk-poznan/ui';

const App = () => {
  <AppShell theme="dark">
    <Button>This is a button in dark theme</Button>
  </AppShell>;
};
```
